--------------------------------------------------------------------------------------------
Модуль GTUAnalyzer
Статический анализатор — не хранит состояние. Принимает мгновенные показания шести датчиков.
--------------------------------------------------------------------------------------------

Класс тестов           | Что проверяется
_______________________________________________________________________________________
TestModeClassification | Классификация по rpm, пограничные значения
TestAnomalyDetection   | Норма, одиночные и множественные аномалии, TRANSITION/EMERGENCY
TestIntegration        | Реальные значения из симулятора
TestLimitsStructure    | Полнота и корректность таблицы MODE_LIMITS
----------------------------------------------------------------------------

Тестовая версия. Заглушки: авторизация (любые данные принимаются), журнал событий (не реализован), смена пароля (не реализована).
